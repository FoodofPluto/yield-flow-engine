# FuruFlow v5

This build upgrades the Streamlit app toward a more product-like DeFi analytics interface.

## Highlights
- fixed low-contrast text inside select controls like Protocols, Strategy Type, Sort by, and Choose a Pool
- opportunity cards for top pools
- richer Plotly charts for protocol, chain, and pool views
- protocol badge styling for a more frontend-like feel
- persistent watchlist layer using Streamlit session state
- compact scanner table with CSV export

## Deploy
Replace your repo's `app.py` and `requirements.txt`, then commit and push to the branch connected to Streamlit Cloud.
